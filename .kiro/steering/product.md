# Product Overview

This is a loan management system (LMS) for managing client information and loan contracts. The system consists of:

- **Admin Portal**: Web-based interface for managing clients, loan contracts, and transactions
- **Client Portal**: LINE LIFF integration for client-facing features
- **REST API**: Backend service handling business logic and data persistence

## Core Features

- Client management (personal information, contact details)
- Loan contract management (principal, interest rates, terms, installments)
- Transaction tracking
- Authentication and authorization (Better-auth)
- Admin dashboard with analytics

## Business Domain

The system manages:
- **Clients**: Individual borrowers with citizen ID, personal details, and contact information
- **Loans**: Loan contracts with principal amounts, interest rates, terms, installment schedules, and status tracking (Active, Closed, Overdue)
- **Transactions**: Payment and transaction history
